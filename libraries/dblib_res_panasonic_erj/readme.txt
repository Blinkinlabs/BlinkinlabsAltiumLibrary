Footprints are re-generated using the IPC footprint wizard template. The wizard has trouble opening the .xls file, so save it as a .csv before opening it in the wizard.

The following fields are important for BOM generation:
'Case/Package' should be something like '0402'
'PartNumber' should have the manufacturer part number
'Manufacturer' should have the manufacturer name
